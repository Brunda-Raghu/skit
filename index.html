About the Project

The Smart Proctoring System is an AI-based solution designed to monitor students during online examinations. It uses computer vision and machine learning techniques to detect suspicious activities such as multiple faces, absence from camera, and abnormal behavior.

Start Demo

Key Features

Face Detection

Detects presence or absence of student face in real time.

Multiple Face Alert

Identifies more than one face during examination.

Head Movement Tracking

Monitors abnormal head movements using AI.

Violation Logging

Automatically records suspicious activities.

System Workflow

  1. Student logs into the examination portal
  2. Webcam starts automatically
  3. AI monitors student behavior
  4. Violations are detected and logged
  5. Admin reviews examination report
